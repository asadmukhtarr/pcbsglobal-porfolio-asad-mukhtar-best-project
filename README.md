# 📌 Project Overview

**PCBs Global ERP / CRM** is a modern enterprise resource planning and customer relationship management system developed for a US-based PCB manufacturing company.

This project represents the complete modernization of a legacy Core PHP application that had been running in production for several years. The objective was not only to redesign the software but also to rebuild the entire application using a modern Laravel ecosystem while preserving the company's existing production database.

The system manages every major business operation including quotations, purchase orders, customer management, vendor management, invoices, shipments, stock, reports, credits, and much more.

Rather than creating a simple CRUD application, this project focuses on solving real business problems by improving maintainability, performance, scalability, and overall user experience.

---

- Component-Based Development
